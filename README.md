# [Weather demo](https://github.com/ShivamKumarJha/android_weather_demo)
A demo Android application for Weather written in Kotlin, follows MVVM repository pattern.  
[Signed APK](https://github.com/ShivamKumarJha/android_weather_demo/blob/master/app/release/app-release.apk)

### Components Used:
* MVVM architecture
* LiveData
* Kotlin coroutines
* Retrofit
* OkHttp + interceptor
* Stetho to monitor response in debug builds.
* Dagger 2
* Recycler View
* Constraint layout
* JUnit4
* Espresso
* min SDK is 16
